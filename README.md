# DotNetSample - Order Processing Application

This scaffold contains an ASP.NET Core Web API (DotNet 8) for a simple order processing system.

Projects:
- Api: ASP.NET Core Web API
- Core: Domain models and interfaces
- Infrastructure: EF Core (SQLite) and repository implementations
- Worker: Background order processor
- Tests: xUnit tests

Run:
- dotnet build
- dotnet run --project Api
